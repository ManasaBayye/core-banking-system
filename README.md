# 🏦  Banking System

## Overview

The Core Banking System is a Java-based application developed as a learning project to understand and practice core Java programming concepts. It simulates real-world banking operations such as account creation, deposits, withdrawals, balance enquiry, and transaction history.

## Features

- Create Account
- Customer Management
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Transaction History
- PIN Validation
- Input Validation

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Collections Framework
- Exception Handling
- Git
- GitHub

## Learning Objectives

- Core Java
- OOP Concepts
- Banking System Design
- Clean Coding Practices
